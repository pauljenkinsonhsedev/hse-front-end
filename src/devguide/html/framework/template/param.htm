Parameters

Template parameters indicate values for controlling content in pages based on a template.

How to use

Parameters can be found at the bottom of every page that has the hse.gov.uk template applied. To use the parameters you will need to change the value of the parameter and then update the page in Dreamweaver.

The following list of values can be used, see the components and regions pages for more information.

When to use

A change in value to a paremeter can be used for changes to layout, editable content regions, style and using a template component.

Parameters
Name Type Values Status
bodyClass text home, news, guidance, abouthse, freeUpdates, contact active
layout text betacontentpage, betatwocol active
columnLayout number 1, 2 active
platform text v5 active
rdfa boolean true, false review
welsh boolean true, false active
homepage boolean true, false review
weblite boolean true, false review
feedback boolean true, false active
contentType text default review
notificationBar boolean true, false active
printGuide boolean true, false review

Body class

Adds the value of the parameter as a css class to the HTML body tag

This parameter is used to changed the active style of the main menu, a value of “guidance” would highlight the guidance item in the main menu.

Layout

Modifies page to display different content regions and change layout of page, betacontentpage is the default value for this parameter.

Column Layout

Modifies the layout of the page, there are 2 values to choose from:

Name Parameter value Description
Full width layout 1 Full width layout which can be used with V5 CSS layout
2 Column layout that includes content region for secondary navigation 2 Standard content page with secondary navigation, V4 css box layout ported to v5 and can be used on all 2 column pages.

Column layout parameter is only applicable to the betacontentpage layout.

Platform

A conditional parameter for changing page header, Default value is v5.

RDFA

Adds RDFA attributes to page

Welsh

Adds Welsh language attribute to page content if value set to true.
