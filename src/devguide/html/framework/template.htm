Template

Apply template to page

This method is best when you are creating a single page from scratch.

  1. Open page in Dreamweaver
  2. From the main menu go to Tools > Templates > Apply template to page...
  3. Select template you want to apply to page
  4. Save page

Change template

This method is used when applying a different template to pages or the entire local site.

  1. Close all pages open in Dreamweaver
  2. Select files you want to apply template to in the file window or select entire local site
  3. From the main menu go to Find > Find and Replace in Files
  4. In the find box, enter the existing template name e.g v5-hse-1.2.0.dwt
  5. In the replace box, enter the name of the template you want to apply e.g v5-hse-1.2.1.dwt
  6. Select replace all

The template you want to apply needs to be in the templates folder of your local site
