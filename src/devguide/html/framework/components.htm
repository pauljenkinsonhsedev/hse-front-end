Notification banner

The notification banner displays on all pages below the main menu.

When to use

Use a notification banner to tell the user about something they need to know about, but thatâ€™s not directly related to the page content.

How to use

To use the notification banner you will need to have access and permission to upload to HSE.GOV.UK, this is needed to upload a JSON file that contains the notification message information.

The JSON file is named global-message.json and can be found in the website root assets folder secureroot/assets/ajax/global-message.json

To display the banner, set the JSON property of 'active' to true and enter a HTML paragraph as a JSON string in the 'message' property.

JSON file example

JSON

A JSON string must be double-quoted, according to the specs, so you don't need to escape '. If you have to use special character in your JSON string, you can escape it using \ character.

See this list of special character used in JSON :

\b Backspace (ascii code 08)
\f Form feed (ascii code 0C)
\n New line
\r Carriage return
\t Tab
\" Double quote
\\ Backslash character

User satisfaction

Each webpage on HSE.GOV.UK has the option to add user feedback at the top of the webpage footer, this is optional and can be displayed using template parameters.

When to use

User feedback is currently applied to all pages and is an important part of measuring user satisfaction on HSE.GOV.UK.

How to use

Set the feedback param value to true and run a template update on the page.

If you are applying the user feedback component to multiple pages, you can run a find and replace, followed by a template update.

Survey

The survey section is an option that can be used in conjunction with the default user feedback survey. When the survey is enabled, if a user selects yes or no they will then be prompted to complete a survey.

How to add survey

To include a survey you will need to add a value to the feedbackClass parameter, as this suvery is experimental there is currently only one value which is 'feedback-beta'.
