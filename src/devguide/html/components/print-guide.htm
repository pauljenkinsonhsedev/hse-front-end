Print guide

Sometimes a user may want to print off guidance as a collection of pages pertaining to an entire section of guidance.

This feature will create a PDF document which the user can download or print the PDF from their browser.

This can easily be achieved by creating print buttons referencing a JSON file containing the pages you want to collate along with some metadata.

Creating a print guide

Simply add this html to your page and create a .json file in the root folder 'hse.gov.uk/print-guides/YOUR_JSON_FILE.json'.

Example HTML

<button class="btn btn-print btn-primary print-guide" data-print-guide="simple-health-safety-risk.json">Print this guide</button>
<button class="btn btn-download btn-primary download-guide" data-print-guide="simple-health-safety-risk.json">Download this guide</button>

Example JSON

https://www.hse.gov.uk/print-guides/simple-health-safety-risk.json

{
          "pages": [
            "./index.htm",
            "./steps-needed-to-manage-risk.htm",
            "./risk-assessment-template-and-examples.htm",
            "./more-detail-on-managing-risk.htm"
          ],
          "metadata": {
            "title": "Managing risks and risk assessment at work",
            "keywords": "risk, assessment, simple, record, potential, harm, measures, details, companies, employer, employee, health, safety",
            "description": "As an employer, you must make a 'suitable and sufficient assessment' of risks to your employees' health and safety, and risks to others because of your work."
          }
      }

Legal information

HSE's Legal information is automatically appended to the end of the document produced. Legal information can be updating using the legal-information.json file.

https://www.hse.gov.uk/print-guides/legal-information.json

{
            "heading": "Legal information",
            "content": [
                "This guidance is issued by the Health and Safety Executive. Following the guidance is not compulsory, unless specifically stated, and you are free to take other action. But if you do follow the guidance you will normally be doing enough to comply with the law. Health and safety inspectors seek to secure compliance with the law and may refer to this guidance."
            ]
        }
