Data attributes

Markup

The charts use data attributes and html markup to define the data, making it accessible for those not using Javascript in their browsers. Data attributes on the container provide additional information for determining the data of the chart and cosmetic appearance.

Chart data attributes
Data attribute Options Required Description
data-chart-type
  • pie
  • donut
  • line
  • bar
  • barstacked
  • column
  • columnstacked
  • arearange
 Yes Determines what type of chart to display.
data-color-scheme
  • red
  • green
  • blue
  • purple
  • brown
  • pink
 No

Sets the colour scheme for the chart. Defaults to an array of colours for luminance and contrast.

To default to a colour contrast scheme leave null.

Single colour schemes will display in varying shades. To be used when only two sets of data.
data-chart-units
  • units
  • percentage
  • percentage-calc
  • hundreds
  • thousands
  • millions
  • billions
  • celsius
  • fahrenheit
  • pounds
 No Determines how values are displayed.
data-decimal-place
  • 0
  • 1
  • 2
  • 3
  • 4
 No Determines how many decimal places. Defaults to 0.
data-chart-title text No Adds a title.
data-chart-subtitle text No Adds a sub title.
data-chart-description text No Adds a description.
data-xaxis-text text No Text for explaining x-axis.
data-yaxis-text text No Text for explaining y-axis.
data-xaxis-firstlast
  • true
  • false
 No Displays only the first and last value on the x-axis.
data-area-range-title text No Generally used to declutter charts that have many data points, for example a range of years.
