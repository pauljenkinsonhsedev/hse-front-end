Area-range chart

An area range chart requires a little more data than our other charts. This can be broken down into three areas:

  • the average
  • the lower bound
  • the upper bound

As you can see in the example we are applying some extra class names to define these columns of data as follows:

  • .series-range-axis (the category)
  • .series-range-average (the average)
  • .series-range-low (the lower bound)
  • .series-range-high (the upper bound)
