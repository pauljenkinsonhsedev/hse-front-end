Missing data

Missing data is easily represented in a bar or column chart, however for line and area-range charts some extra work has to be done behind the scenes to make this work. For ease when developing we can enter a value of ‘0’ to represent the null data.

Currently for area-ranges we must still supply the estimated data for the low band and high band and only supply zero values for the average.

Naturally the tooltip won’t display where we have null data as expected.
