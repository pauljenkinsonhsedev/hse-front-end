Image attributes

Important: All content images should be contained inside a <figure> element with the class .v5-figure to make use of v5 styles detailed on this page.

Height

The height of an image is calculated with CSS auto height. This CSS attribute will overide the HTML height attribute and ensure that the image height is correct.

Width

The width of an image will be set by the width attribute in the image element HTML.

Aspect ratio

The reccomended aspect ratio for images is 16:9.

Full width responsive image

Wrap the image element inside a <figure> element with a class of .v5-figure and .full-width.

You don’t need to remove the height and width in the HTML.

Code example

<figure class="v5-figure full-width">
  <img alt="Inspector Calls" height="394" width="700" src="assets/images/hse-inspector.jpg">
</figure>

Caption

The HTML <figcaption> represents a caption or legend describing the rest of the contents of its parent <figure> element.

To add a caption to your image, place a <figcaption> element inside the relative <figure> element.

Code example

<figure class="v5-figure full-width">
  <img alt="Inspector Calls" height="394" width="700" src="assets/images/hse-inspector.jpg">
  <figcaption>This is a caption for an image</figcaption>
</figure>

When to use

When there is a need to provide context to the image, avoid unnecessary decoration. Only use images if there’s a real user need.

Accessibility

From an accessibility viewpoint, captions and alt text have distinct roles. Captions benefit even people who can see the image, whereas alt text provides the same functionality as an absent image. Therefore, captions and alt text shouldn't just say the same thing, because they both appear when the image is gone. Try turning images off in your browser and see how it looks.

Fixed width image

Code example

<figure class="v5-figure">
 <img alt="HSE Inspector" height="auto" width="450" src="assets/images/hse-inspector.jpg">
</figure>

Fixed image with caption

Fixed images can also have captions but make sure the first element below the figure element has a class of .utilclearfloats.

Caption width

Below demonstrates an issue with fixed image width and captions, if the caption width is larger than the image width it will not be automatically constrained to the image width.

To fix this issue add a class of .fix to the <figcaption> element.

Small caption

Caption text is small in this example and does not push the <figcaption> element beyond the width of the image.

Large caption

Alternative text

The alt attribute should be included in all img elements.

Its value is supposed to be a textual description of the image, for use in situations where the image cannot be seen/displayed or takes a long time to render because of a slow internet connection.

Decorative images

If you must include decorative images in your HTML, add a blank alt="". If the image isn't part of the content, a screen reader shouldn't waste time reading it.

Content

If your image provides significant information, provide the same information in the main text which everybody can see or include a brief alt text.

Text

You should not use images that include text.

Links

If the img element is inside <a> element, either, write it inside the same <a> element, or inside the image's alt attribute.
