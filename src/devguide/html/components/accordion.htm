Accordion (JS)

The accordion component lets users show and hide sections of related content on a page.

No default opened panel & ability to open multiple panels at a time.

How to use

Accordion container

The accordion uses data attributes, to create an accordion you will need to assign the following data attributes to a <div> element:

  • data-aria-accordion
  • data-multi
  • data-transition
  • data-default="none"

Headings and panels

Each section is made up of a heading and it's corresponding panel, they must be placed inside the parent accordion <div> element.

Choose a heading element (Making sure to keep the heading structure correct), then assign a data attribute of data-aria-accordion-heading. It is best practice to add an id to the heading element, to allow for anchor links to this section.

For your section content you will need to create a <div> element directly below the corresponding heading and assign a data attribute of data-aria-accordion-panel.

Repeat this process to add multiple sections to the accordion.

When to use

Only use an accordion if there’s evidence it’s helpful for users to:

  • see an overview of multiple, related sections of content
  • show and hide those sections as needed
