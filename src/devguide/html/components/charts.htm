Charts

HSE charts uses the HighCharts plugin. As a page may contain multiple chart types, to make things faster to load and easier to write, there is a javascript layer to deal with the various options you can supply for various types of charts. No additional javascript is required to populate a chart unless it is to support a new type of chart not already covered here.

Chart types:

  • Pie
  • Donut
  • Line
  • Bar
  • Bar stacked
  • Column
  • Column stacked
  • Area-range

Markup

The charts use data attributes and html markup to define the data, making it accessible for those not using Javascript in their browsers. Additionally HighCharts can be read by screen readers and also can be tabbed making it accessible. Data attributes on the container provide additional information for determining the data of the chart and cosmetic appearance.

Chart with plot bands

Some data along the plot may require further explanation or supporting text. For example if data between a certain range of years was not recorded, it can be shown that this data was only estimated.

Chart with confidence intervals

Sometimes an author may want to represent data which is not entirely accurate. Thus confidence intervals can be a good way to indicate how confident the accuracy of each node of data is.

Chart with missing data

Missing data is easily represented in a bar or column chart, however for line and area-range charts some extra work has to be done behind the scenes to make this work. For ease when developing we can enter a value of ‘0’ to represent the null data.

Currently for area-ranges we must still supply the estimated data for the low band and high band and only supply zero values for the average.

Naturally the tooltip won’t display where we have null data as expected.
