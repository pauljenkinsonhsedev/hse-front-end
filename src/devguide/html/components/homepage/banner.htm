Homepage banner

How to use

The homepage banner component must be used in conjunction with the v5 layout. It is designed to be the same width in desktop view as a .twoThird column.

Within the two third column you will firstly need to create an <a> tag element with a href path to a page on hse.gov.uk.

You then need to create one <div> element within the <a> tag element with the class name .homepageBannerImage and another div element with the class name .homepageBannerInner.

Banner image and text

.homepageBannerImage is the container for your banner image and .homepageBannerInner is a container for the banner text.

Heading

The homepage banner heading should always use a <h2> tag.

Sub heading

If a sub heading is required, use the <p> tag. The font size is set to 1.375rem in CSS to distinguish as a sub heading and not a standard paragraph.

When to use

The homepage banner should only be used on the hse.gov.uk homepage to promote important content.

Responsive images

The HTML <picture> element is used to offer alternative versions of an image for different display/device scenarios.

The <source> media attribute specifies a media condition (similar to a media query) that the user agent will evaluate for each <source> element.

If the <source> media condition evaluates to false, the browser skips it and evaluates the next element inside <picture>.

Source: media

The homepage banner requires two <source> elements, one with a media condition for desktop devices and the other for mobile devices.

Desktop Mobile
(min-width: 980px) (max-width: 980px)

Source: srcset

The srcset attribute is used to offer list of possible images based on size.

Each image requires a URL path to the homepage banner image file.

<srcset=“assets/v4-homepage/images/corona-hand-sanitizer-mobile.png, assets/v4-homepage/images/corona-hand-sanitizer-mobile-2x.png 2x“>

A pixel density descriptor can be used to specify a higher quality image for devices with retina displays, this is indicated with a 2x or 3x at the end of the image URL.

Scale factor Devices
x1 Standard-resolution devices
x2 Retina displays and all other high-resolution iOS devices
x3 iPhone X, iPhone 8 Plus, iPhone 7 Plus, and iPhone 6s Plus

Generating Design Assets

The following design assets are recommended when creating a homepage banner:

  • Desktop standard resoluton banner
  • Mobile standard resoluton banner
  • Desktop retina banner (x1)
  • Mobile retina banner (x1)

Accessibility

Do not use images of text if you expect the text to be read by the user, unless it’s necessary such as in a logo or brand name.
