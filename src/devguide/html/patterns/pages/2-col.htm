Two Column page

A two column page uses secondary navigation, which in desktop view is aligned to the left in the first column, the second column contains the editable region for content.

When to use

A two column page should be used when the content is related to a group of pages and needs to be linked with secondary navigation.

How to use

To use the one column layout, set the layout parameter to 'betacontentpage' and the columnLayout parameteer to 'two'

Layout

Box layout can be used with the two column page but it is only intended for creating microsite homepages.

Components

All content componenets can be used in the two column page and the following component patters:

  • Complementary content

    Complementary content should always be included at the bottom of the two column page layout, no further content should be placed below it.

  • Subscribe

    A subscribe box can be included anywhere after the page <h1> title

  • Page contents box

    Page contents box has to be included before all page content and after the page <h1> title
