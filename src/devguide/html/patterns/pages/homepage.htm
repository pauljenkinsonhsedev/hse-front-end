Microsite homepage (box layout)

Microsite homepages are built using the box layout and feature secondary navigation.

The box layout homepage has been replace by the new v5 homepage.

When to use

Microsite homepages are to be used to provide a homepage for a topic or industry microsite. They should be included as part of the content design process when creating a new microsite that is made up of 2 column pages.

How to use

To use the microsite homepage, set the layout parameter to 'betacontentpage' and the columnLayout parameteer to 'two'

You will then need to design your homepage using the box layout, while the homepage is designed to be flexible, the following best practices should be adhered to:

  • Include a <h1> tag before start of box layout
  • Boxes should be used to contain short descriptors with links to microsite pages
  • Do not overload boxes with content, a seperate page should be created and linked to from the homepage.
  • Do not include decorative images
  • Use boxes with more link rather than linking from paragraph text
  • Boxes can be used to group related links e.g. Frequently Asked Questions
  • Prioritise content by order of rows, first row should have the most important content.
  • A microsite homepage should only be used once per microsite.
  • Do not include content above or below the homepage box layout
