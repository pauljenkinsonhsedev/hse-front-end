Demo page

When to use

A to Z lists help users find the page they want by bringing together links to lots of pages about one broad subject. They work best when users know the name of the thing they're looking for and they expect to find it under that subject, e.g. industries.

How to use

An A-Z page requires two elements, one for navigation and the other for the list of page links.

The navigation is a <ul> element with the class of .alphabetList, contained within a <nav> element with the class .boxAtoZ.

The list of pages needs to be contained within a div element with the class .listAtoZ, <h2> headings are used to define the letter group for each list. Each letter requires a seperate ul element and should have more than one list item. If it only has one list item, you should use a <p> element instead with a class of .linkList.

Each heading in the list of pages should have a unique id, you can simply use the letter of each group as the id. The navigation element uses in page links for each of the letters to jump the user to the list of pages.

Code example

A-Z navigation


<nav class="boxAtoZ" aria-label="A to Z Navigation">
 <ul class="alphabetList">
 <li><a href="#a">A</a> </li>
 <li><a href="#b">B</a> </li>
 <li><a href="#c">C</a> </li>
 <li><a href="#d">D</a> </li>
 <li><a href="#e">E</a> </li>
 <li><a href="#f">F</a> </li>
 <li><a href="#g">G</a> </li>
 <li><a href="#h">H</a> </li>
 <li class="empty"> <span class="emptyLetter">I</span></li>
 <li class="empty"> <span class="emptyLetter">J</span></li>
 <li class="empty"> <span class="emptyLetter">K</span></li>
 <li><a href="#l">L</a> </li>
 <li><a href="#m">M</a> </li>
 <li class="empty"> <span class="emptyLetter">N</span></li>
 <li><a href="#o">O</a> </li>
 <li><a href="#p">P</a> </li>
 <li><a href="#q-az">Q</a> </li>
 <li><a href="#r">R</a> </li>
 <li><a href="#s">S</a> </li>
 <li><a href="#t">T</a> </li>
 <li class="empty"> <span class="emptyLetter">U</span></li>
 <li class="empty"> <span class="emptyLetter">V</span></li>
 <li><a href="#w">W</a> </li>
 <li class="empty"> <span class="emptyLetter">X</span></li>
 <li class="empty"> <span class="emptyLetter">Y</span></li>
 <li class="empty"> <span class="emptyLetter">Z</span></li>
 </ul>
</nav>

A-Z list


<div class="listAtoZ">

 </ul>

 <p class="linkList"><a href="../../biocides/index.htm" title="Biocidal products are used in a variety of industries to control unwanted organisms, such as animals, insects, bacteria, viruses and fungi">Biocides</a> </p>
