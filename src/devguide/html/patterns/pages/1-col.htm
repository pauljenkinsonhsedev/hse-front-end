One Column page

A one column page uses the full width of the content container and does not include the secondary navigation menu.

One column demo page

When to use

You can use the one column page when there is no need for the page content to be linked to another by secondary navigation.

How to use

To use the one column layout, set the layout parameter to 'betacontentpage' and the columnLayout parameteer to 'one'

Layout

No layouts can be used with the one column page and it is strictly for use with content and content components

Components

All content componenets can be used in the two column page and the following component patters:

  • Complementary content

    Complementary content should always be included at the bottom of the two column page layout, no further content should be placed below it.

  • Subscribe

    A subscribe box can be included anywhere after the page <h1> title

  • Page contents box

    Page contents box has to be included before all page content and after the page <h1> title
