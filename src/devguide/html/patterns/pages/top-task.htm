Top task

When to use

A top task page should be used when there is a group of pages that are related and can be sequenced to make it easier for the user (e.g. An employer) to understand and meet their Health and Safety requirements.

How to use

The top task page is made up of the following elements:

  1. Navigation
  2. Article
  3. Pagination
  4. Complementary content

Navigation

Top task pages include navigation in the form of an ordered list at the top of each page, immediately below the <h1> title. The list should contain the core pages of the top task.

Article

The article region contains the main content of the page, all content components can be used in this region.

Pagination

Pagination is included on each top task page and is always placed inside the article region as the last component with no content or components below it.

Complementary content

The complementary content is always included in the asideBottom editable content region and should relate to the page content.

The resources box should always come before the related content box on a top task page.
