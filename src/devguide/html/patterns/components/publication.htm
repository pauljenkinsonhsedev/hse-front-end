Publication box

The publication box is made up of a box component, description list and unordered list, providing the main details of the publication.

Code example

When to use

The publication box should be used when you are creating a 2 column landing page for a HSE Books publication.

How to use

The publication box should be placed immediately after the page <h1> title or sub title if included.

The description list will need to be placed inside a box component, a single list item or unordered list can then be used to create links to the PDF or HSE Books landing page.

HSE Books

The following details can be included in the description list of a HSE Books publication:

  • Date of publication
  • ISBN
  • Series code

Links to the free PDF copy of the publications and a link to the HSE Books product page need to be provided at the bottom of the publication box.
