Subscribe box

Subscribe forms are displayed on the website to give users the option to enter their email address and sign up for HSE e-bulletins.

Subscribe

When to use

The inclusion of a subscribe box should be decided by the content author and digital performance analyist. The e-bulletin the user subscribes to should relate to the page content.

How to use

A subscribe box is made up of form components and the box component. The form contains an input field for the user to enter their email address, once submitted they will be taken to GovDelivery and prompted to enter their email address again to subscribe to the e-bulletin service.

To create a subscribe form you will first need to create a <form> element and give it an id and an action attribute, the value of the action attribute will be the following string:

"https://public.govdelivery.com/accounts/UKHSE/subscriber/qualify

E-bulletin topic

A hidden input form is required to enter the topic ID of the e-bulletin. Create an input field with the following attributes:

 <input id="topic_id" name="topic_id" type="hidden" value="UKHSE_***">

Get the relevant topic id from the table of e-bulletin codes and include the code in the value attribute.

Styling form elements

Depending on what page and layout you're using, the subscribe form will need to be adjusted to fit. To adjust the width of the input element you should refer to the form components input sizes.

Regardless of page or layout, the subscribe form should always be contained inside a box component.

Code example

<div class="box">

<form class="subscribe" action="https://public.govdelivery.com/accounts/UKHSE/subscriber/qualify">
	
 <fieldset class="fieldset">
	 
  <legend class="fieldset__legend">
  <h3 class="fieldset__heading">Subscribe</h3>
  </legend>

<!-- Include hidden input and unique e-bulletin value -->
  <input id="topic_id" name="topic_id" type="hidden" value="UKHSE_125"> 

  <div class="form-group">
   <label class="label" for="email">Subscribe for free news and updates on health and safety topics and industries</label>
   <input class="input input-email input-width-l" type="email" id="email" name="email" placeholder="Your email address" required title="Please enter your email address">
   <button class="btn btn-cautionary" name="commit" type="submit">Subscribe</button>
  </div>

 </fieldset>
	
</form>

</div>
