Cards

Cards demo page

When to use

The ability to present small snippets of simplified information that enable the user to get a birdâ€™s eye view of the content on offer.

Cards replace the box layout method of creating microsite homepages and are designed to provide navigation to top task pages.

How to use

Cards can be created usings unordered lists and applying the class of .multiCol to the parent div of the list element.

To specificy the number of columns, you need to add an aditional class of .col2 or .col3 to the same parent div element.

To remove the default list style, you will need to add the class of .columnsList

Code example

<div class="columnFull border noBoxPadding">
  <div class="column multiCol col3">
    <ul class="columnsList">
      <li>
        <h3><a href="#">List item</a></h3>
        <p>List item descriptor</p>
      </li>
