Start (call-to-action) button

Experimental: This component is currently experimental because more research is needed to validate it.

Start button

When to use

Use a start (call-to-action) button to emphasise a link to an online service, a form or another publication where a person must carry out an action in order to complete a task.

Do not use it to link to something that a person may need to do or know.

Unless linking directly to an online service, link to the document landing page where possible.

It should instruct the person to do something so begin the text with a verb.

There will usually be just one start button on a page but if the page is long consider inserting another at the end of the page if appropriate.

How to use

To use the start button you need an <a> element with the following form button classes applied to it: .btn .btn-primary .btn-start

Accessibility

A <button> element is not used for the start button because you are not triggering an action, e.g. submitting a form or opening a dialog box.

Code example

<a class="btn btn-primary btn-start">Start button</a>
