Complementary content

Complementary content is used to designate a supporting section that relates to the main content, yet can stand alone when separated.

When to use

Complementary content should be specified by the content author, it should relate to the content of the page and not be the same on every page in a microsite just because it shares the same industry or topic.

Related content

Related content is a list of links to content on hse.gov.uk that is related to page content.

How to use

Related content is made up a box component and an unordered list with markers. It should always contain a <h2> element with the text 'Related content'.

Resources

Resources are primarily digital assets that are not webpages but will be of use to the user or relvant to the content page.

The most common type of resource on HSE.GOV.UK is a HSE books publication.

How to use

Resource boxes are created the same way as related content and see also but they can include thumbnails of the resource.

To use the resources list style you will need to use a class of .itemThumbBook with a <ul> element.

Each list item must contain a thumbnail image and a <span> element which contains the title of the resource.
