Page contents

Contents

  • When to use
  • How to use

Page contents is in-page navigation that uses anchor links to link to content that you would otherwise have to scroll to view.

When to use

A page contents pattern should only be used on pages that require considerable scrolling to view all of the page contents.

How to use

A page contents pattern uses the box component and an unordered list.

You need to first create a container for the contents list, apply the .box class to a <div> element.

Using the HTML achor links method, you then need to add a unique ID to each heading on the page that you will be linking to.

Finally you add your list items to the list in your page contents box, each list item will be a link to an in-page heading. To make the in-page links work, you need to have a # before the heading ID.

Code example

<div class="box">
  <h2>Contents</h2>
  <ul>
    <li><a href="#how-to-use">How to use</a></li>
    <li><a href="#when-to-use">When to use</a></li>
  </ul>
</div>
