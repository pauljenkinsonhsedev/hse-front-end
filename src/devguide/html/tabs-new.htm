Tabs (JS)

The tabs component lets users navigate between related sections of content, displaying one section at a time.

Use tabs to allow users to navigate easily between views within the same context.

How to use

Container

To create a tabs component you need to create a container using a <div> element and assign it a class of .tabContainer.

Panels

To create a tab panel you need to add a <div> inside of the tab container and assign it a class of .tabPanel. It is then best practice to assign it a unique id for use with achor links. The panel content can then be inserted within this <div>.

Panel headings

Each panel requires a heading, insert a heading within the tab panel <div> container to display a heading on the tab. This heading will also be visible in the panel content.

Repeat this process to add multiple tab panels.

When to use

Tabs can be a helpful way of letting users quickly switch between related information if:

  • your content can be usefully separated into clearly labelled sections
  • the first section is more relevant than the others for most users
  • users will not need to view all the sections at once
