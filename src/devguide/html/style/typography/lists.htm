Lists

Use lists to make blocks of text easier to read, and to break information into manageable chunks.

List types

  • Bulleted
  • Ordered
  • Alphabetically ordered
  • Roman numerical ordered
  • Tick
  • Cross
  • Bullet links
  • Marker links
  • Split
  • Description

Accessibility

  • Lists containing no links – use a regular <ul> tag
  • Lists used in content areas, eg with anchor points – use a regular <ul> tag
  • Lists containing more than one item linking to a page outside of the current one – use <ul class="linkList">
  • One link, that would usually sit with a bullet point, that points to a page outside the current one – use <p class="linkList"> instead of a <ul> (this class will be getting changed at a later date)

Description list

The HTML <dl> element represents a description list. The element encloses a list of groups of terms (specified using the <dt> element) and descriptions (provided by <dd> elements).

Date of publication:
February 2021
ISBN:
9780717667420
Series code:
N/A

Code example

<dl>
  <dt>Date of publication:</dt>
  <dd>February 2021</dd>
  <dt>ISBN:</dt>
  <dd>9780717667420</dd>
  <dt>Series code:</dt>
  <dd>N/A</dd>
</dl>
