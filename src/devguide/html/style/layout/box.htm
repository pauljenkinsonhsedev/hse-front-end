Box layout

The box layout used in the V4 framework has been carried to the V5 framework and made responsive.

  • It is best practice to use all box layouts in a div with the class .columnFull
  • Box layout is intended to be used with the betacontenpage and 2 column layout
  • Boxes can span a maximum of 3 columns
  • Box layout is primarily used to create microsite homepages
  • Using equal heights is recommended for all layouts
  • Box layout can have multiple rows
  • The last box of each row needs to have a class of .boxLast
  • The box layout is designed to contain snippets of content that link through to a page containig further related content.
  • All boxes span 100% of the page container when in mobile view and stack from left to right.

Container

To create a layout, you will need to create a container with a div element and apply a class of .columnFull to the div.

The container spans the full with of the page container, 940px wide. It becomes a fluid width container at the desktop breakpoint of 980px.

<div class="columnFull">
 <!-- Column layout -->
</div>

Columns

The box layout uses a simple 3 column grid, each box can span from 1 to 3 columns.

Each box is a div with a class of .box and a choice of box column class.

Box column class
number of columns class
1 box, box1col
2 box, box2col
3 box, box3col (Using just the primary class of primary will give the same result)

box box1col

box box1col

box box1col boxlast

box box2col

box box1col and boxLast

box or box3col

Code example

<div class="columnFull">
	  <div class="box box1col">
	    <h2>box box1col</h2>
	  </div>
	  <div class="box box1col">
	    <h2>box box1col</h2>
	  </div>
	  <div class="box box1col boxLast">
	    <h2>box box1col boxlast</h2>
	  </div>
	  <div class="box box2col">
	    <h2>box box2col</h2>
	  </div>
	  <div class="box box1col boxLast">
	    <h2>box box1col and boxLast</h2>
	  </div>
	  <div class="box box3col">
	    <h2>box or box3col</h2>
	  </div>
	</div>

Split box

A div can be used inside a box to split a box in half, using the class .boxSplit. It works as a seperator and no content should be placed inside this div.

Split boxes will merge together in mobile view to form a single box, take this into consideration when designing your layout.
