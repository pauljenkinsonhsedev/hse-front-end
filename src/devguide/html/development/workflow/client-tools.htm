Git-client management tools

If you are new to Git then learning Git commands in the CLI is recommended in order to give you a thorough understanding of how it works. A great place to learn about Git is git-scm.com which is an excellent guide and has useful documentation.

However it can sometimes become convoluted and time consuming unpicking commmits when things go wrong. Therefore we recommend using a client which has Git-flow also integrated into the client.

SourceTree is a free and easy to use product.
