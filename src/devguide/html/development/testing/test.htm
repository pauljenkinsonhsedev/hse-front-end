Green (.btn .btn-primary)

To be used on 'primary' actions for example 'Continue' or 'Save'.

Red (.btn .btn-cautionary)

To be used on 'cautionary' actions for example 'Cancel' or 'Close'.

Yellow (.btn .btn-important)

To be used on 'important' actions for example 'Hide message'.
